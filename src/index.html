How to Install

Windows Installer (.exe)

  1. Download and run SuperQAT Setup
  2. Follow the installer prompts
  3. Open SuperQAT from the Start Menu — it runs alongside Word

Mac Installer (.dmg)

  1. Open the downloaded .dmg file
  2. Drag SuperQAT to your Applications folder
  3. Open it from Applications — it runs alongside Word

Word Add-in (.dot) — Adds Dropdown to Word's Ribbon

  1. Download SuperQAT.dot
  2. Windows: Press Win+R, paste %APPDATA%\Microsoft\Word\STARTUP and hit Enter. Drop the file there.
  3. Mac: In Finder, press Cmd+Shift+G, paste ~/Library/Group Containers/UBF8T346G9.Office/User Content/Startup/Word/ and drop the file there.
  4. Restart Word
  5. Click the Add-ins tab — you'll see the QAT Command dropdown with all 1,067 commands
